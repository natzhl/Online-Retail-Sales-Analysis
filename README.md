# Online-Retail-Sales-Analysis

Анализ продаж интернет-магазина на основе Kaggle датасета “Online Retail Dataset”.

_____________________________
Цель проекта — выявить сезонность продаж, определить причины падения выручки и найти ключевые товары и страны, формирующие основную часть дохода компании.

Очистка данных выполнена в Python (pandas): удалены возвраты, нулевые цены и строки без CustomerID, добавлены признаки выручки и месяца. 
Дальнейший анализ проведён в Excel.
